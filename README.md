# ISA STANDARD ATMOSPHERE CALCULATOR

## DESCRIPTION
Calculates atmospheric pressure, temperature, and density based on the International Standard Atmosphere model. 
Input: altitude in meters. 
Output: ISA values.

## WHY DID I BUILD IT?
Built to practice C++ while studying aeronautical engineering.

## HOW TO TRY IT?
If you want to try it. You only have to :
1.- Download calculadora.exe and atmosfera.dll files.
2.- Double clic on calculadore.exe. The the terminal will open asking you for the height in meters.

## WHAT ARE THE FORMULAS I USED IT?
1.- T = T0 - L * h
2.- P = P0 * ((T1 / T0) ^ (g / (L * R)))
3.- rho = P / (R * T)
